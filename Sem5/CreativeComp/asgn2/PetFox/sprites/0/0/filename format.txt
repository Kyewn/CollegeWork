Filename format:
pet<growth value><mood value><hygiene value>.png

Growth value:
1 - Small
2 - Medium
3 - Big

Mood value:
This value reflects both hunger and mood status bar, if either status bars go below minimum value for Good mood, sprite will change
1 - Good
2 - OK
3 - Not good

Hygiene value:
1 - Clean
2 - Dirty
